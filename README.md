# Hotel Reservation System

A Java Swing-based Hotel Reservation System developed as part of the CodeAlpha Java Programming Internship.

## Features

* View available hotel rooms
* Book rooms
* Cancel bookings
* Room categorization (Standard, Deluxe, Suite)
* Booking status management
* GUI using Java Swing

## Technologies Used

* Java
* Java Swing
* OOP Concepts
* ArrayList
